Eric Fassiau — Site Vitrine

Site vitrine réalisé pour présenter l’activité artistique de Eric Fassiau.

Ce projet s’inscrit dans ma reconversion professionnelle Artiste 3D → Développeur Web, avec pour objectif de concevoir un site moderne, performant et facilement maintenable.

Le projet met l’accent sur :

une expérience utilisateur claire

une mise en valeur visuelle du contenu

un design responsive

des bonnes pratiques d’intégration web
